Questions, Bugs & Feature Requests
***********************************

If you have any bugs, feature requests, or questions about how to install or use the application please post on the |messageBoard|. 

Frequently Asked Question
=========================
    #. My Recorder does not contain after analysis is completed.
    
        By default, recorder will write to file after the wipe (Python: ops.wipe() ) command is called unless the default is overridden.

    #. What is the Unit for OpenSees?
    
        OpenSees has no unit, please use a consistent input-units for your input.
