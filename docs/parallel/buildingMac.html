Building Parallel Versions¶

  1. Download latest stable release from OpenMPI, which at time of writing was openmpi-4.0.2. We downloaded into Downloads: openmpi-4.0.2.tar.gz

cd ~/Downloads
tar zxBf openmpi-4.0.2.tar.gz
cd openmpi-4.0.2/
./configure --prefix=/usr/local/openmpi
make -j 4
sudo make install
usr/local/openmpi/bin/mpirun --version

     mpirun (Open MPI) 4.0.2

     Report bugs to http://www.open-mpi.org/community/help/

Note

gfortran was previously installed on our machine.
