3.1.5.23. CoulombDamper Material¶

This command is used to construct the uniaxial CoulombDamper material producing elastic stiffness and friction force.

uniaxialMaterial CoulombDamper $matTag $Tangent $FrictionFoce -tol $tol -numFlipped $numFlipped -reduceFc -dampOutTangent $dampOutTangent

Argument

Type

Description

$matTag

integer

integer tag identifying material.

$Tangent

float

the tangent stiffness for strain.

$FrictionFoce

float

the friction force opposite to strain rate.

$tol

float

the tolerance to check if strain rate is small.

$numFlipped

integer

the number of times that the strain rate is flipped before adjusting friction force.

-reduceFc

string

reducing friction force if strain rate is flipped several times.

$dampOutTangent

float

giving a constant damping tangent if strain rate is flipped several times.
